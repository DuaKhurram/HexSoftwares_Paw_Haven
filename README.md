# 🐾 HexSoftwares_Paw_Haven

## 📌 Project Overview
**Paw Haven** is a luxury animal charity website that evolved from a premium pet day care service. It showcases the journey from day care to full-fledged animal rescue and rehabilitation center based in Karachi.

## 🎯 Features
- Responsive luxury design with gold/rose gold theme
- Interactive donation system with amount selection
- Volunteer sign-up with unique role-based logos
- Impact counter displaying rescued animals, adoptions, etc.
- Instagram integration with @haven_for_paws
- Wishlist with enlarged custom logos
- Newsletter subscription
- Smooth scroll navigation

## 🛠️ Technologies Used
- HTML5
- CSS3 (Custom properties, Flexbox, Grid, Animations)
- JavaScript (ES6)
- Font Awesome 6
- Google Fonts (Quicksand, Playfair Display)
